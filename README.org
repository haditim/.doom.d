* .doom.d
My personal doom settings with focus on Python
* Usage
Install [[https://github.com/hlissner/doom-emacs][doom emacs,]] then run

#+BEGIN_SRC shell -n
mv .doom.d .doom.d.bak
git clone https://github.com/haditim/.doom.d
.emacs.d/bin/doom refresh
#+END_SRC
